# Changelog

## 0.1.0

- Initial release.
- Dot-grid mesh system (`DotGridWidget`, `DotGridController`, `DotGridGroup`).
- Themed widgets: `TermosButton`, `TermosBackButton`, `TermosNavBar`, `TermosSegmentedSelector`, `TermosSwitch`, `TermosTextField`, `TermosTimePicker`, `TermosCrt`, `TermosLoadingIndicator`.
- Slider family: `TermosSlider`, `TermosFloatingSlider`, `TermosContinuousSlider`, `TermosDetailedSlider`.
- Custom painters: `ReactiveStarfieldPainter`, `GlowTopBorderPainter`, `ScanlinesPainter`, `VerticalStarfieldPainter`, `EdgeGlowPainter`.
- `TermosThemeData` with `dark()` / `light()` factories and `heavyEffectsEnabled` toggle.
- Accessibility: `Semantics` on all interactive widgets.
